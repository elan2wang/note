###Mac下VIM配置
vim配置文件：`/usr/share/vim/vimrc`  
部分配置提示：  
<pre>
syntax on               " syntax highlighting
set hlsearch            " highlight the last searched term
set ruler               " show the cursor position
set ai                  " auto indenting
</pre>
